Description

The Package Journey component displays a timeline of the package's delivery journey. It shows each status update with location, timestamp, and description.

API

Property Type Description
journeyPoints JourneyPoint[] Array of journey points displayed in the timeline
showAllPoints boolean Controls whether all points are displayed or just recent ones
toggleShowAll() function Method to toggle between showing all points and showing only recent ones

JourneyPoint Interface

export interface JourneyPoint {
  location: string;
  timestamp: string;
  description: string;
  icon: string;
  status: 'completed' | 'pending';
}
