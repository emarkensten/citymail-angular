Bottom Sheet Component

The Bottom Sheet component is a modal that slides up from the bottom of the screen, taking only as much vertical space as needed for its content.

Example

Bottom Sheet Title

This is a bottom sheet that animates in from the bottom of the screen.

It only takes up as much vertical space as needed for its content.

Usage

<!-- Modal Structure -->
<div class="modal fade" id="myBottomSheet">
  <div class="modal-dialog modal-auto-height">
    <div class="modal-content bottom-sheet bottom-sheet-container">
      <div class="modal-header">
        <h5 class="modal-title">Sheet Title</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        <!-- Sheet content -->
      </div>
    </div>
  </div>
</div>

<!-- JavaScript -->
const modal = new bootstrap.Modal(document.getElementById('myBottomSheet'));
modal.show();

CSS Classes

Class Description
bottom-sheet Applies rounded corners to the top of the sheet
bottom-sheet-container Handles the slide-in animation from bottom
modal-auto-height Makes the sheet take only as much space as needed (up to 85vh)

Animation Details

/* Bottom sheet animation and sizing */
.modal.fade .bottom-sheet-container {
  transform: translateY(100%);
  transition: transform 0.2s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

.modal.show .bottom-sheet-container {
  transform: translateY(0);
}

/* Override Bootstrap's modal size for auto-height */
.modal-auto-height {
  max-height: 85vh;
  margin: 0;
  position: absolute;
  bottom: 0;
  width: 100%;
}
